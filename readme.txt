== Description ==

= Products Stock Manager with Excel for WooCommerce plugin gives the ability to update your stock inventory and prices with the power of excel  =

The Plugin extends WooCommerce and allows you to update easily your Simple products based on SKU or ID stock and prices , automatically update with CRON job / Sync (**[PRO Version](https://extend-wp.com/product/products-stock-manager-excel-woocommerce "Products Stock Manager with Excel for WooCommerce PRO")**) .
It is using a predefined **excel template** that you can easily locate in its **Settings** Page or otherwise use its **Export** functionality to get the right Template. In the Pro version you can also predefine the mapping in plugin settings page.
Moreover, you will get **Stock Report** for your Business Management Purposes
In the (**[PRO Version](https://extend-wp.com/product/products-stock-manager-excel-woocommerce "Products Stock Manager with Excel for WooCommerce PRO")**) you can automate the Process of Update with Excel from Remote Location eg Google Spreadsheet, update and export Variable products and variations.

= How I can benefit from Stock Manager inventory  =

* Fast Update by ID or SKU simple products with the Power of Excel
* Locate the ID and SKU of your products easily by using the Export feature in Excel in the right format
* Stock Report for your Stock Management Purposes

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins` directory and unzip, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to WooCommerce / Stock Manager

== Frequently Asked Questions ==

= How to how to build my excel template? =
Download a sample excel file from Update Screen of plugin settings. The fields updated are predefined and follow the exact excel template

= How to update stock and prices via excel? =
You can either go to Export screen first and get an excel of your products. Other option is to download a sample excel file from Update Screen of plugin settings. The fields updated are predefined and follow the exact excel template

= Can I automatically update at certain time ? =
This is available in the [PRO Version](https://extend-wp.com/product/products-stock-manager-excel-woocommerce "Products Stock Manager with Excel for WooCommerce PRO")

= Can I update Product variations? =
This is available in the [PRO Version](https://extend-wp.com/product/products-stock-manager-excel-woocommerce "Products Stock Manager with Excel for WooCommerce PRO")
